# Overview
This project is my exersice to implement multi provider login using 
PHP and [opauth](https://github.com/opauth/opauth) library.

# Step-by-step
1.  include this to composer:
	```
	{
	    "require":{
	        "opauth/opauth": "*",
	        "opauth/facebook": "*",
	        "opauth/google": "*",
	        "opauth/twitter": "*"
	    }
	}
	```
